# Portfólio — Rafael Guimarães

Portfólio profissional desenvolvido com React, TypeScript e Vite. O conteúdo está disponível em português e inglês, com tema claro e escuro e layout responsivo.

## Desenvolvimento local

```bash
npm install
npm run dev
```

## Validação

```bash
npm run build
npm run preview
```

O deploy para o GitHub Pages é realizado automaticamente pelo workflow em `.github/workflows/deploy.yml` após alterações na branch `main`.
